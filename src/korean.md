## Grammar

### Subject Markers

- Indicate the subject of a sentence.
- Used for emphasizing the subject or introducing new information.
- Sentence form: Subject + Subject Marker + Verb/Adjective

| Subject Marker | Usage                                                                | Example                             |
| -------------- | -------------------------------------------------------------------- | ----------------------------------- |
| 이 (i)         | Used when the subject ends with a consonant. Example: 책 + 이 = 책이 | 책이 있어요. (There is a book.)     |
| 가 (ga)        | Used when the subject ends with a vowel. Example: 사과 + 가 = 사과가 | 사과가 있어요. (There is an apple.) |

### Object Markers

- Indicate the direct object of a verb.
- Sentence form: Subject + Subject Marker + Object + Object Marker + Verb

| Object Marker | Usage                                                               | Example                          |
| ------------- | ------------------------------------------------------------------- | -------------------------------- |
| 을 (eul)      | Used when the object ends with a consonant. Example: 책 + 을 = 책을 | 책을 읽어요. (I read a book.)    |
| 를 (reul)     | Used when the object ends with a vowel. Example: 사과 + 를 = 사과를 | 사과를 먹어요. (I eat an apple.) |

### Topic Markers

- Indicate the topic of a sentence.
- Used for general statements, background information, or contrast.
- Sentence form: Topic + Topic Marker + Comment

| Topic Marker | Usage                                                              | Example                                               |
| ------------ | ------------------------------------------------------------------ | ----------------------------------------------------- |
| 은 (eun)     | Used when the topic ends with a consonant. Example: 책 + 은 = 책은 | 책은 재미있어요. (As for the book, it's interesting.) |
| 는 (neun)    | Used when the topic ends with a vowel. Example: 사과 + 는 = 사과는 | 사과는 맛있어요. (As for the apple, it's delicious.)  |

### Location Markers

- Indicate where something happens or exists.
- Sentence form: Location + Location Marker + Verb/Adjective

| Location Marker | Usage                                                                        | Example                                                |
| --------------- | ---------------------------------------------------------------------------- | ------------------------------------------------------ |
| 에 (e)          | Used for a specific location or point in time. Often with 있어요 and 없어요. | 집에 있어요. (I am at home.)                           |
| 에서 (eseo)     | Used for the location where an action takes place.                           | 학교에서 과학을 공부해요. (I study science at school.) |

### Verb Conjugation: Present Tense

- Use 아요 if the verb stem ends with ㅏ or ㅗ.
  - 가다 → 가요
  - 보다 → 봐요
  - 오다 → 와요
  - 찾다 → 찾아요
- Use 어요 if the verb stem ends with any other vowel.
  - 먹다 → 먹어요
  - 마시다 → 마셔요
  - 배우다 → 배워요
  - 기다리다 → 기다려요
- If the verb stem ends with 하, use 해요.
  - 공부하다 → 공부해요
  - 운동하다 → 운동해요
- Examples
  - 친구를 만나요. (I meet a friend.)
  - 책을 읽어요. (I read a book.)
  - 영화를 봐요. (I watch a movie.)
  - 커피를 마셔요. (I drink coffee.)

### Also / Too

- 도 (do) means “also” or “too”.
- It can replace subject, object, and topic markers.
- Sentence form: Subject/Object/Topic + 도 + Verb/Adjective
- Examples
  - 저는 학생이에요. 미나도 학생이에요. (I am a student. Mina is also a student.)
  - 책이 있어요. 사과도 있어요. (There is a book. There is also an apple.)

### How to use numbers in Korean

- Sino-Korean numbers are used for **dates, money, addresses, phone numbers**, and counting above 99.
- Native Korean numbers are used for **counting people, objects, hours, and age**.
- Examples
  - 오늘은 2024년 6월 15일이에요. (Today is June 15, 2024.)
  - 제 전화번호는 010-1234-5678이에요. (My phone number is 010-1234-5678.)
  - 학생이 세 명 있어요. (There are three students.)

### Date and Time Expressions

- Use Sino Korean numbers for years, months, days, minutes, and seconds.
- Use Native Korean numbers for hours.
- Date form: `Year (SN)` + 년 + `Month (SN)` + 월 + `Day (SN)` + 일
- Time form: 오전/오후 + `Hour (KN)` + 시 + `Minute (SN)` + 분 + `Second (SN)` + 초
